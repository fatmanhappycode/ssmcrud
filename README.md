# ssmcrud(员工增删改查系统)
### 使用了Spring Spring-MVC MyBatis框架，采用Javaconfig配置
### 使用H2数据库作为测试数据库，具有关闭持久化功能的数据库，加载sql文件创建样例，重启服务器时恢复会使用前的状态，也可以在web.xml中将test改为development可以激活开发生产的Mysql数据库
### Boostrap框架做前端页面，MyBatis逆向工程生产代码，Ajax，json做交互
